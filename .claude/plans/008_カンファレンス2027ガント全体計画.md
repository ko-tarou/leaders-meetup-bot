# 008: カンファレンス2027 ガント/タスク管理 内製実装（全体計画）

方針（本人承認済み・2026-07）:

- モジュラーモノリス採用（ADR-0009）。`src/modules/gantt/` が第 1 号。全面リファクタはしない。
- API ファースト運用（ADR-0010）。運用書き込みは `scripts/lmb-api.mjs`（本番 API + ADMIN_TOKEN）経由。DB 直 SQL 書き込みは廃止。
- イベント階層: 「カンファレンス2027」イベント（新 CLI で作成）> `gantt_tracker` アクション。
- MVP = 設計比較の案 A: tasks へ 4 列追加（team/phase/wbs/progress_pct・後方互換）+ task_dependencies + config(teams/phases/summaryGroups)。
- 画面 3 つ: ガント（テーブル + SVG タイムライン + バードラッグ + 依存矢印）/ 全体サマリー（18 項目・サーバ導出）/ 月別ビュー。
- データ移行: v7 Excel の 60 タスクを CLI 経由 API で投入（SQL 直入れ禁止）。データ JSON はリポジトリにコミットしない（public repo のため）。
