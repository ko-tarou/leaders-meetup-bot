# 001-4 PR4: 日程調整 — 投票作成・送信

## 概要

スラッシュコマンド `/meetup` で日程調整の投票を作成し、Slackチャンネルに送信する機能を実装する。

## フロー

1. ユーザーが `/meetup 2026-04-20 2026-04-27 2026-05-04` と入力
2. Bot が候補日をパースし、D1 に poll + poll_options を保存
3. Bot が Block Kit の投票メッセージをチャンネルに送信
4. メッセージの ts を poll レコードに保存

## 実装内容

### src/services/poll.ts（新規）
- createPoll(db, meetingId, channelId, dates[]): Poll作成 + オプション作成 + Slack送信
- UUID生成には crypto.randomUUID() を使用

### src/routes/slack.ts（更新）
- /commands ハンドラで `/meetup` コマンドを処理
- コマンドテキストから候補日をパース（スペース区切り）
- poll.ts の createPoll を呼び出し
